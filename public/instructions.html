API Basejump: Timestamp microservice

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a Unix timestamp or a natural language date (example: December 3, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or a Unix timestamp, it returns null for those properties.

Example usage:

https://timestamp-ns.herokuapp.com/December%203,%202016
https://timestamp-ns.herokuapp.com/1480723200000

Example output:

{ unix: 1480723200000, natural: "December 3, 2016" }
